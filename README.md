### To Run (For windows):
#### If node is not installed, install it.
1. Open your terminal. This can be found by simply searching 'Terminal' in your applications
2. Run the following commands, pressing enter after each one:
  -  ```powershell -c "irm https://community.chocolatey.org/install.ps1|iex"```
  -  ```choco install nodejs --version="24.14.1"```
  - ```node -v``` This should print "v24.14.1"
  - ```npm -v``` This should print "11.11.0"
4. If either C or D does not return the expected series, seek help at https://nodejs.org/en/download or reach out

#### Running the Website
1. Download Repo as a ZIP file. This is located under the code button dropdown
2. Extract the file from the zip file by double clicking on it.
3. Right click on the folder that was extracted and select "open in terminal"
4. First, run ```npm install```
5. Then, run: ```npm run dev```

### View In Browser
View the site by visiting ```localhost:5173``` in your browser of choice
